# FlightOps Data Platform / Plataforma de Datos de Operaciones Aeroportuarias

## ES — Descripción

Proyecto end-to-end de data engineering para ingerir, transformar y modelar datos de operaciones aeroportuarias y meteorología en datasets listos para análisis usando Python, SQL, BigQuery, OpenSky y Open-Meteo.

## ES — Objetivo del proyecto

Construir un pipeline de datos que ingiera información de vuelos y clima, la transforme en tablas analíticas y publique un dataset final a nivel de aeropuerto-hora.

## ES — Alcance inicial

El MVP del proyecto se centrará en:
- aeropuertos de España
- datos de llegadas y salidas desde OpenSky
- enriquecimiento meteorológico con Open-Meteo
- modelado analítico en BigQuery
- tabla final de operaciones por aeropuerto y hora

## EN — Description

End-to-end data engineering project to ingest, transform, and model airport operations and weather data into analytics-ready datasets using Python, SQL, BigQuery, OpenSky, and Open-Meteo.

## EN — Project goal

Build a data pipeline that ingests flight and weather data, transforms them into analytical tables, and publishes a final airport-hour level dataset.

## EN — Initial scope

The MVP will focus on:
- Spanish airports
- arrivals and departures data from OpenSky
- weather enrichment with Open-Meteo
- analytical modeling in BigQuery
- final airport-hour operations dataset
